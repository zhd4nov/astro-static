# Стековые инженерные правила

## Назначение

Эта директория хранит инженерные правила для конкретных стеков и языков.

Базовые стек-независимые правила находятся в `docs/engineering-standards.md`.

## Правила

- `astro-react.md` - правила для Astro-проекта с React-компонентами.
- `typescript.md` - правила использования TypeScript.

## Порядок добавления

1. Добавляй стековый документ после утверждения стека или языка.
2. Не дублируй базовые правила из `docs/engineering-standards.md`.
3. Фиксируй только правила, которые можно применить при реализации.
4. Если правило можно проверить инструментом, добавь его в спецификацию разворачивания проекта.
