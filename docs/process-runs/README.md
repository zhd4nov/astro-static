# Журналы процесса

## Назначение

Эта директория хранит журналы процесса для ретроспективы, сервисного режима или явного запроса.

Правила использования журналов указаны в `docs/agent-workflow.md` и `docs/process-improvement.md`.

## Шаблон

Используй `docs/process-runs/template.md`.

## Имена файлов

Формат:

`RUN-YYYY-MM-DD-short-name.md`

Пример:

`RUN-2026-05-01-first-site-slice.md`

## Реестр

Общий реестр журналов находится в `docs/process-runs/index.md`.
