# Результаты ревью

## Назначение

Эта директория хранит файловые ревью, когда для них есть основание.

Правила ревью находятся в `docs/review-standards.md`.

## Шаблон

Используй `docs/reviews/template.md`.

## Имена файлов

Формат:

`REVIEW-NNNN-rX-short-name.md`

Пример:

`REVIEW-0003-r1-home-specialists-slice.md`
