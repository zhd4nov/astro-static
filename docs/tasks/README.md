# Задачи

## Назначение

Эта директория хранит `TASK` для обычных и сложных задач.

Правила создания и обновления задач находятся в `docs/task-management.md`.

## Шаблон

Используй `docs/tasks/template.md`.

## Имена файлов

Формат:

`TASK-NNNN-short-name.md`

Пример:

`TASK-0001-select-site-stack.md`

## Реестр

Общий реестр задач находится в `docs/tasks/index.md`.
